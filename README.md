# [React] Todo Application

## 구현 내용

- 할일 목록 보여주기
- 할일 추가하기
- 할일 삭제하기
- 할일 체크하기
- 필터링 기능 (전체 / 완료 / 미완료)
- 다크모드 구현
- localstorage 이용해 DB 저장

## 적용 개념

- Context API
- PostCSS
- localStorage
